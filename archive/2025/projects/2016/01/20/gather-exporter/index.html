Gather exporter

Gather exporter allows you to export/backup the collections you have entered in Wikipedia (English, Hebrew, etc) into a few different formats.

This web app built with React.js polls the Wikipedia API and downloads your collections, showing them on a table.

If you want to download them, it’ll give you the option to download a zip file generated on the browser containing the data in different formats (JSON, WikiText, YAML, HTML) so that you can have your lists backed up.
